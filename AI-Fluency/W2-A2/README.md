# Foundations Track (FL-01): Prompt Engineering Iteration Log

## 📌 Assignment Overview
This repository contains the completed assignment for the **FL-01 Foundations Track (Prompt Engineering)**. The project documents the process of taking a "naive" single-line prompt and systematically refining it through six distinct iterations using industry-standard prompt engineering techniques. 

The primary task chosen for this log focuses on transforming raw, unstructured task specifications into structured, production-ready project documentation and evaluation matrices.

---

## 🎯 Objectives
* **Master Core Prompt Engineering Techniques:** Implement and observe the specific impact of System Roles, Context & Motivation, Few-Shot Exemplars, Structural XML Constraints, and Chain-of-Thought (Step Decomposition).
* **Quantify Output Improvements:** Document exact output changes and failure points across iterations rather than just noting prompt edits.
* **Cross-Model Evaluation:** Conduct an empirical comparison between **Claude 3.5 Sonnet** and **ChatGPT (GPT-4o)** using the final engineered prompt.
* **Build a Reusable Asset:** Create a context-decoupled, production-ready prompt template usable by anyone for similar tasks.
